Welcome!

Glad to have you here! If you enjoy aviation, designing, and want to make Infinite Flight better, then you've come to the right place!

Infinite Flight contains many airports worldwide, and our mission is to make sure they are as accurate as possible. We're looking for people to help with creating accurate layouts, with taxiways, taxi lines, ATC frequencies and parking spots.

Getting started



Watch this video to help you get started

Before you can begin editing airports, a few things need to be done:

  • Create a GitHub account and fork the repo.
  • Install a version of X-Plane and download World Editor (WED)
  • Clone the repo

Getting started is very easy. The video above will give you a full, guided, step-by-step guide on how to prepare for editing airports. Here are the steps involved:

  1. Create a GitHub account, by entering your details on the GitHub website. Remember these details, they'll come in handly later!
  2. Go to http://github.com/carmichaelalonso/InfiniteFlight-Airports, and select the 'Fork' button in the top right corner. This will make a copy of the airports under your account.
  3. Download the GitHub app (For Mac, For Windows).
  4. Install a copy of X-Plane. Skip this step if you already have a DVD/Steam copy, otherwise you can download the free demo from x-plane.com/
  5. Install a copy of World Editor from here.
  6. Login to the GitHub app, then tap the '+' button on the top left corner, select the 'Clone' tab, and select "InfiniteFlight-Airports". Save this to somewhere you'll remember later.
  7. Open WorldEditor, and select the folder in which you installed X-Plane to. Create a new scenery package, and call it the name of the airport you want to update.
  8. A new window should appear. Go to File - Import from apt.dat
  9. Find the folder in which you cloned the repo to earlier. Select the airport you want to edit, and import the apt.dat
That's all there is to it! You're good to go!
